# Portfólio de Sinthian Marques

Este é o meu **portfólio online**, onde compartilho minha trajetória como **Desenvolvedora Back-end Java** e minha evolução em **AWS Foundations**.  

Aqui você pode explorar minhas habilidades, certificações e projetos, tudo documentado de forma clara e organizada.

---

## Sobre o portfólio

O portfólio foi construído com **HTML, CSS e Bootstrap**, combinando design clean com navegação intuitiva. Ele é dividido em seções principais:

- **Sobre mim:** Um resumo da minha trajetória profissional, experiências anteriores em suporte técnico e análise operacional, e meu objetivo atual como desenvolvedora.
- **Skills:** Apresentação das minhas hard e soft skills, destacando minhas competências técnicas e interpessoais.
- **Certificações:** Lista completa dos cursos e certificações que concluí, incluindo FIAP, ENAP e DevDojo.
- **LinkedIn:** Acesso direto ao meu perfil profissional para contato e networking.

---

## Tecnologias utilizadas

- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Design:** Layout responsivo e interativo
- **Ícones e fontes:** Font Awesome, Google Fonts
- **Documentação e aprendizado:** GitHub para versionamento e registro da evolução técnica

## Objetivo

O portfólio foi criado para mostrar minha **jornada de aprendizado**, meus **conhecimentos em desenvolvimento back-end** e minha **busca por oportunidades na área de tecnologia**. Ele também serve como espaço para registrar minhas experiências práticas com **cloud computing (AWS)** e automações.
